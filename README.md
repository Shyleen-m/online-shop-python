Beauty & Cosmetics E-Commerce System (Python)
Overview

This project is a command-line based e-commerce application developed in Python, simulating an online shopping platform focused on beauty and cosmetic products.

Features
Product inventory management
Shopping cart functionality
Add and remove product operations
Checkout and total price calculation
User interaction through a menu-driven interface
Input validation and error handling
Technologies Used
Python
Object-Oriented Programming (OOP)
Project Structure
product.py – Product data structure and inventory details
cart.py – Shopping cart operations and calculations
user.py – User management and cart interaction
main.py – Main application workflow and menu system
Purpose

This project was built to practice object-oriented programming, user interaction handling, and basic e-commerce system logic in Python.
